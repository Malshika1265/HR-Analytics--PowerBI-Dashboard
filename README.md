# HR-Analytics--PowerBI-Dashboard
## 📌 Project Overview
This Power BI dashboard provides insights into workforce trends, employee demographics, and attrition. It helps HR professionals monitor key metrics and make data-driven decisions.

## 🎯 Objectives
- Analyze employee demographics
- Monitor employee attrition
- Compare department performance
- Identify workforce trends
- Support HR decision-making

## 🛠️ Tools & Technologies
- Power BI Desktop
- DAX
- Power Query
# 📈 Dashboard Features

## KPI Cards

- Total Employees
- Active Employees
- Attrition Count
- Attrition Rate (%)
- Average Age
- Average Experience

---

## Interactive Filters

- Department
- Age Group

---

## Visualizations

### Employee & Attrition Analysis

- Attrition by Department (Donut Chart)
- Attrition by Salary Slab (Bar Chart)
- Attrition by Gender (Pie Chart)
- Attrition Trend by Total Experience (Line Chart)

### Workforce Analysis

- Department-wise Employee Count
- Age Group Distribution
- Job Role vs Job Satisfaction Matrix
